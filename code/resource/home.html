The basic command is /7s N where N is the number of dice you will roll!

You can add several commands to the roll such as:
By adding e every 10 will be withdrawn!
By adding b N you will add a bonus N to each die rolled
By adding s N you will change the rise threshold to N (default is 10)
By adding v N will add N despicable dice to the roll
